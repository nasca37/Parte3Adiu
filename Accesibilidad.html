Accesibilidad

Esta página web ha sido creada con la intención de realizar la práctica de Accesibilidad de la asignatura Aplicacions Distribuïdes a Internet i Interficies d'Usuari. De esta forma nos hemos centrado en crear una pagina sencilla pero contiene diferentes aspectos que la mayoría de páginas web contempla como son el texto, las imagenes, graficas, videos, tablas y formularios. Todos estos aspectos cumplen las normas de Accesibilidad del nivel AA que se declara en la normativa de WCAG 2.0. Para cercionarnos de este hecho hemos recurrido a una página de comprobación automatica recomendada en la bibliografia de la asignatura. Esta página es conocida como achecker. A continuación se deja una tabla que para cada uno de los archivos html muestra el icono de la certifación y el enlace para descargar el pdf que corresponde a la certificación de cada página.

Obviamente como ya conocemos no es posible asegurar de forma completa que una pagina web asegura todas las funcionalidades necesarias para que sea lo más accesible posible para todo tipo de usuarios. Por ese motivo a continuación revisaremos algunos de los aspectos que no se pueden verficar de forma automatica y requieren por tanto de un toque humano.
